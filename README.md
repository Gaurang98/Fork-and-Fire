# 🍕 [Fork & Fire](https://fork-and-fire.vercel.app/)
🔗 [Website](https://fork-and-fire.vercel.app/)

A modern, responsive food delivery web application built with React.js, featuring a beautiful UI, smooth animations, and comprehensive functionality.

## ✨ Features

### 🏠 Core Functionality
- **Modern Landing Page** with animated hero section
- **Interactive Menu** with category filtering (Pizza, Burgers, Pasta, Desserts)
- **Shopping Cart** with add/remove/update quantity functionality
- **User Profile** with editable information and order history
- **Responsive Design** that works on all devices

### 🎨 UI/UX Features
- **Smooth Animations** using Framer Motion
- **Text Animations** (Split text, fade-in, slide effects)
- **Gradient Backgrounds** and modern color scheme
- **Mobile-First Design** with hamburger menu
- **Profile Dropdown** with navigation options

### 💰 Pricing
- **Indian Rupee (₹) Currency** with realistic pricing
- **Category-wise pricing** ranging from ₹199 to ₹699

## 🚀 Tech Stack

- **Frontend**: React.js (Functional Components with Hooks)
- **Routing**: React Router DOM
- **Animations**: Framer Motion
- **Styling**: CSS3 with CSS Variables
- **Icons**: Unicode Emojis
- **Images**: Unsplash API with fallback system

## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Gaurang98/Fork-and-Fire.git
   cd Fork-and-Fire
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000` (Default)

## 📦 Available Scripts

- `npm start` - Runs the app in development mode
- `npm test` - Launches the test runner
- `npm run build` - Builds the app for production
- `npm run eject` - Ejects from Create React App (⚠️ irreversible)

## 📱 Pages & Components

### Pages
- **Landing Page**: Hero section with call-to-action
- **Home**: Featured items and company highlights  
- **Menu**: Full menu with filtering capabilities
- **About**: Company story, team, and values
- **Contact**: Contact information and form
- **Profile**: User profile management and order history
- **Cart**: Shopping cart with checkout functionality

### Components
- **Navbar**: Responsive navigation with profile dropdown

- **TextAnimations**: Various text animation components
- **Cart**: Shopping cart functionality

## 🎯 Key Features

### Menu System
- **Dynamic Filtering**: Filter items by category
- **Price Display**: All prices in Indian Rupees (₹)
- **Image Fallbacks**: Robust image loading system

### Cart Functionality
- **Add Items**: Add items to cart with quantity tracking
- **Update Quantity**: Increase/decrease item quantities
- **Remove Items**: Remove items from cart
- **Total Calculation**: Real-time total calculation

### Profile System
- **User Information**: Editable profile details
- **Order History**: View past orders with details
- **Account Management**: Profile editing capabilities
